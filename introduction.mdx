# Welcome to Kanagawa Dotfiles

A beautiful, minimalist dotfiles collection designed for Arch Linux users who love Hyprland and want a productive, aesthetically pleasing desktop environment centered around the stunning Kanagawa color scheme.

<img src="/screenshots/desktop.png" alt="Desktop Preview" width="100%" />

## What is This?

Kanagawa Dotfiles is a carefully curated configuration system that transforms your Arch Linux installation into a modern, efficient workspace. Built on top of Hyprland (Wayland compositor), this configuration brings together the best tools in the Linux ecosystem with a cohesive visual design inspired by the Kanagawa wave.

**"Configuración minimalista, productiva y centrada en el esquema de colores Kanagawa."**

## Key Features

<AccordionGroup>
  <Accordion title="🎨 Multi-Theme Support" icon="palette">
    Switch between Kanagawa, Gruvbox, Catppuccin, and Everforest themes instantly using the built-in theme selector. All components update automatically including:
    - Hyprland borders and decorations
    - Waybar status bar
    - Wofi launcher
    - Ghostty terminal
    - Neovim editor
    - VSCodium IDE
    - SwayNC notifications
    - Wlogout power menu
    - Btop system monitor
    - Cava audio visualizer
  </Accordion>

  <Accordion title="⚡ Modern Wayland Stack" icon="bolt">
    Built entirely on Wayland for tear-free graphics, better performance, and modern display features. Uses Hyprland as the compositor for smooth animations and powerful tiling.
  </Accordion>

  <Accordion title="🛠️ Developer-Focused Tools" icon="code">
    Includes modern CLI replacements and productivity tools:
    - `bat` for syntax-highlighted file viewing
    - `eza` for better directory listings
    - `zoxide` for smart directory jumping
    - `fzf` for fuzzy finding
    - `atuin` for shell history
  </Accordion>

  <Accordion title="📦 GNU Stow Management" icon="box">
    All configurations are managed with GNU Stow for clean, symlink-based deployment. Easy to install, update, and remove without cluttering your home directory.
  </Accordion>

  <Accordion title="🚀 One-Command Installation" icon="terminal">
    Simple installation script handles everything automatically. No manual file copying or complex setup required.
  </Accordion>
</AccordionGroup>

## Repository Structure

The repository is organized for easy management with GNU Stow:

```bash
~/dotfiles/
├── config/          # Application configurations → ~/.config/
│   ├── hypr/       # Hyprland compositor settings
│   ├── waybar/     # Status bar configuration
│   ├── ghostty/    # Terminal settings
│   ├── wofi/       # Launcher configuration
│   ├── nvim/       # Neovim configuration
│   └── ...         # Other apps
├── scripts/         # Custom automation scripts → ~/.local/bin/
│   ├── theme-selector
│   ├── theme-switcher
│   └── weather
├── shell/           # Zsh configuration → ~/
│   └── .zshrc
├── screenshots/     # Preview images
└── install.sh       # Installation script
```

<Note>
  Each directory maps to a specific location in your system. The `config/` directory contents are symlinked to `~/.config/`, scripts go to `~/.local/bin/`, and shell configs to your home directory.
</Note>
