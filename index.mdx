## Quick Start

Get your environment up and running in minutes.

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/nixer112/dotfiles.git ~/dotfiles
    cd ~/dotfiles
    ```
  </Step>
  <Step title="Run the installation script">
    ```bash
    ./install.sh
    ```
  </Step>
  <Step title="Select your theme">
    ```bash
    theme-selector
    ```
  </Step>
</Steps>
